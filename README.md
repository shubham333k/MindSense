🧠 MindSense-AI: Intelligent Emotional Intelligence (EQ) & Mental Wellness Companion




🏛️ Project Overview

MindSense-AI is a sophisticated, empathy-driven AI system designed to bridge the gap between technology and mental health. Built on a robust NLP pipeline, this companion goes beyond simple keyword matching to analyze mood, complex emotions, sarcasm, and cognitive patterns. By providing personalized, therapist-style responses and adaptive wellness recommendations, MindSense-AI serves as a 24/7 support system for long-term mental state tracking and emotional resilience.

High-Impact Capabilities

• Sarcasm & Nuance Detection: Unlike basic sentiment tools, MindSense-AI is engineered to understand the nuances of human speech, including sarcasm and mixed emotions.

• Long-Term Wellness Tracking: Utilizes a persistent SQLite-based history store to track mental health trends over time, enabling users to see their progress.

• Adaptive Recommendations: Dynamically suggests personalized coping strategies, meditations, or professional resources based on the detected mental state.




🛠️ Technical Architecture

System Components

1. Sentiment Engine (sentiment.py): The core NLP layer for multi-dimensional emotion extraction.

2. Mental State Analyzer (mental_state.py): Evaluates long-term patterns and cognitive health markers.

3. Therapist Model (therapist_model.py): Generates contextually relevant, empathetic responses.

4. Data Persistence (history_store.py): Manages the mental_history.db for secure, long-term trend analysis.

5. Interface (app.py): A clean, modern Streamlit-based UI for seamless user interaction.

Tech Stack

• Language: Python 3.10+

• AI/NLP: Transformers, NLTK, Scikit-learn

• Framework: Streamlit

• Database: SQLite

• Analytics: Pandas, NumPy
